# Compliance & Regulatory Checklist

## Securities Compliance

### Federal Securities
- [ ] Form D filings (each offering)
- [ ] Regulation D compliance
- [ ] Accredited investor verification
- [ ] Bad actor checks
- [ ] Investment representations

**For each securities issuance verify:**
- Exemption relied upon (Rule 506(b), 506(c), etc.)
- Form D filed within 15 days
- Investor qualifications documented
- No general solicitation (if 506(b))

### State Securities (Blue Sky)
- [ ] State notice filings
- [ ] State exemption documentation
- [ ] Multi-state filing compliance

**Check filing requirements for:**
- State of incorporation
- States where investors reside
- States where company operates

## Data Privacy & Security

### Privacy Compliance
- [ ] Privacy Policy (current)
- [ ] Terms of Service
- [ ] Cookie Policy
- [ ] CCPA compliance (if applicable)
- [ ] GDPR compliance (if applicable)

**CCPA requirements (if >$25M revenue, 50K+ consumers, or 50%+ revenue from data):**
- Notice at collection
- Right to know / access
- Right to delete
- Right to opt-out of sale
- Non-discrimination

**GDPR requirements (if EU data subjects):**
- Lawful basis for processing
- Data subject rights
- Privacy notices
- Data processing agreements
- Cross-border transfer mechanisms

### Security
- [ ] Information security policy
- [ ] SOC 2 report (if applicable)
- [ ] Penetration test results
- [ ] Security incident response plan
- [ ] Data breach history

### Children's Privacy
- [ ] COPPA compliance (if collecting data from <13)
- [ ] Age verification mechanisms

## Industry-Specific Regulations

### Healthcare (if applicable)
- [ ] HIPAA compliance assessment
- [ ] BAA agreements with vendors
- [ ] Security risk assessment
- [ ] HIPAA policies and procedures

### Financial Services (if applicable)
- [ ] State money transmitter licenses
- [ ] SEC/FINRA registrations
- [ ] PCI-DSS compliance (if payments)
- [ ] BSA/AML compliance

### Other Regulated Industries
- [ ] FCC compliance (telecom)
- [ ] FDA compliance (medical devices, food)
- [ ] DOT compliance (transportation)
- [ ] State professional licenses

## Corporate Compliance

### Tax Compliance
- [ ] Federal tax returns (3 years)
- [ ] State tax returns
- [ ] Payroll tax filings
- [ ] Sales tax filings
- [ ] 1099 filings

**Delaware Franchise Tax:**
- Annual report filed
- Franchise tax paid
- Assumed par value method used (usually optimal)

### Employment Compliance
- [ ] Wage and hour compliance
- [ ] EEOC compliance
- [ ] OSHA compliance
- [ ] Workers compensation
- [ ] I-9 compliance

### Environmental
- [ ] Environmental permits (if applicable)
- [ ] Hazardous materials handling
- [ ] Environmental audits

## Government Relations

### Lobbying
- [ ] Lobbying registrations (if applicable)
- [ ] Lobbying disclosures

### Export Controls
- [ ] Export classification (EAR, ITAR)
- [ ] Denied party screening
- [ ] Technology export compliance

### Sanctions
- [ ] OFAC screening
- [ ] Sanctions compliance program

## Compliance Programs

### Policies
- [ ] Code of conduct
- [ ] Anti-bribery/FCPA policy
- [ ] Whistleblower policy
- [ ] Conflicts of interest policy

### Training
- [ ] Compliance training records
- [ ] Anti-harassment training
- [ ] Security awareness training
