# Cap Table & Securities Checklist

## Cap Table Documentation

### Stock Ledger
- [ ] Complete stock ledger / cap table
- [ ] Stock certificates or book entry records
- [ ] Stock transfer records

**Verify:**
- All issuances tie to board/stockholder approvals
- Certificate numbers sequential
- No gaps or missing certificates
- Cancelled certificates accounted for

### Cap Table Reconciliation
- [ ] Current cap table (fully diluted)
- [ ] Pro forma cap table (post-financing)
- [ ] SAFE/note conversion modeling

**Cross-check:**
- Cap table matches stock ledger
- All SAFEs/notes included in pro forma
- Option pool matches plan documents
- Ownership percentages calculate correctly

## Founder Equity

### Founder Stock
- [ ] Founder stock purchase agreements
- [ ] 83(b) elections (filed within 30 days)
- [ ] Proof of payment for founder shares
- [ ] Vesting schedules

**Review for:**
- Vesting terms (standard: 4-year, 1-year cliff)
- Acceleration provisions (single/double trigger)
- Repurchase rights for unvested shares
- 83(b) timely filed (CRITICAL)

### Restricted Stock
- [ ] Restricted Stock Purchase Agreements
- [ ] Restricted Stock Award Agreements
- [ ] Right of First Refusal agreements

## Prior Financings

### SAFE Notes
- [ ] All executed SAFE agreements
- [ ] SAFE side letters (if any)
- [ ] Board consents authorizing SAFEs

**For each SAFE, document:**
- Investment amount
- Valuation cap
- Discount rate
- Pro rata rights
- MFN provisions

### Convertible Notes
- [ ] Promissory notes
- [ ] Note purchase agreements
- [ ] Security agreements (if secured)
- [ ] Board consents

**For each note, document:**
- Principal amount
- Interest rate
- Maturity date
- Conversion terms (cap, discount)
- Qualified financing threshold

### Priced Rounds (if any)
- [ ] Stock Purchase Agreements
- [ ] Investors' Rights Agreements
- [ ] Voting Agreements
- [ ] Right of First Refusal / Co-Sale Agreements
- [ ] Certificates of Designation

## Equity Incentive Plans

### Option Plan
- [ ] Equity Incentive Plan document
- [ ] Board approval of plan
- [ ] Stockholder approval of plan
- [ ] Plan amendments

**Verify:**
- Shares reserved under plan
- Shares granted vs. available
- Evergreen provisions (if any)

### Option Grants
- [ ] Stock option agreements (all grants)
- [ ] Board consents for each grant
- [ ] 409A valuation reports
- [ ] Exercise notices (for exercised options)

**For each grant, verify:**
- Grant date matches board consent
- Exercise price â‰¥ 409A FMV at grant
- Vesting schedule documented
- Cliff and acceleration terms

### 409A Valuations
- [ ] Initial 409A report
- [ ] Subsequent 409A reports (every 12 months or material event)
- [ ] Methodology documentation

**Verify:**
- Valuation current (within 12 months)
- Grants made at or above 409A FMV
- No backdating of grants

## Securities Compliance

### Federal Filings
- [ ] Form D filings (each offering)
- [ ] Form D amendments
- [ ] Accredited investor questionnaires

### State Filings (Blue Sky)
- [ ] State notice filings
- [ ] State exemption documentation
