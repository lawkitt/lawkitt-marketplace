# Intellectual Property Checklist

## IP Ownership & Assignment

### Founder IP Assignment
- [ ] Founder IP assignment agreements (pre-incorporation work)
- [ ] Technology assignment agreements
- [ ] Invention assignment clauses in employment agreements

**CRITICAL: Verify assignment of:**
- Code written before incorporation
- Product designs and specifications
- Brand names, logos, domain names
- Customer lists and databases
- Trade secrets and know-how

### Employee IP Assignment
- [ ] PIIA (Proprietary Information and Inventions Assignment) for all employees
- [ ] Confidentiality agreements
- [ ] Invention disclosure forms

**Verify:**
- All current employees have signed PIIAs
- PIIAs signed before starting work
- Prior inventions properly excluded

### Contractor IP Assignment
- [ ] Contractor/consultant agreements with IP assignment
- [ ] Work-for-hire agreements
- [ ] Assignment of all deliverables

**For each contractor who touched product:**
- Written agreement exists
- IP assignment clause present
- Work-for-hire designation (if applicable)
- No retained license back to contractor

## Registered IP

### Patents
- [ ] List of all patents (issued and pending)
- [ ] Patent applications
- [ ] Assignment records (USPTO)
- [ ] Maintenance fee records
- [ ] Prosecution history

**For each patent/application:**
- Owner is the company (not individual)
- Inventors properly listed
- Assignments recorded with USPTO
- Maintenance fees current

### Trademarks
- [ ] List of registered trademarks
- [ ] Trademark applications
- [ ] Registration certificates
- [ ] Specimens of use

**For each trademark:**
- Owner is the company
- Registration current
- Proper use in commerce

### Copyrights
- [ ] Copyright registrations (if any)
- [ ] Software copyright notices

### Domain Names
- [ ] List of all domain names
- [ ] Domain registration records
- [ ] Verify registrant is company

## Technology Stack

### Software
- [ ] List of all proprietary software
- [ ] Source code repository access
- [ ] Technical architecture documentation

### Open Source
- [ ] Open source audit/inventory
- [ ] License compliance review
- [ ] Copyleft license identification (GPL, AGPL, etc.)

**Review for:**
- GPL/AGPL code in proprietary product
- License compatibility issues
- Attribution requirements
- Distribution obligations

### Third-Party Software
- [ ] List of licensed software
- [ ] License agreements
- [ ] SaaS/cloud service agreements
- [ ] API terms of service

**Verify:**
- Licenses permit intended use
- No assignment restrictions
- Fees current

## IP Agreements

### Licenses In
- [ ] Inbound license agreements
- [ ] Technology licenses
- [ ] Content licenses

### Licenses Out
- [ ] Outbound license agreements
- [ ] Customer license terms (in ToS, MSA)
- [ ] Open source contributions

### IP-Related Disputes
- [ ] Cease and desist letters (sent or received)
- [ ] IP litigation
- [ ] DMCA takedown notices
- [ ] Patent assertion entity contacts

## Trade Secrets

- [ ] Trade secret identification
- [ ] Protection measures documentation
- [ ] NDA inventory (customers, partners, etc.)
- [ ] Employee exit procedures
