# Employment & Founders Checklist

## Founder Arrangements

### Founder Agreements
- [ ] Founder/Co-founder agreement
- [ ] Roles and responsibilities documentation
- [ ] Decision-making protocols

**Review for:**
- Equity split rationale
- Departure scenarios
- IP assignment provisions
- Non-compete/non-solicit terms

### Founder Employment
- [ ] Founder employment agreements or offer letters
- [ ] At-will status (or term of employment)
- [ ] Compensation terms
- [ ] Benefits

### Founder Equity
- [ ] Stock purchase agreements
- [ ] Vesting schedules
- [ ] Acceleration provisions
- [ ] 83(b) elections

**Key terms to verify:**
- Standard 4-year vest, 1-year cliff
- Single vs. double trigger acceleration
- Repurchase rights
- Post-termination exercise periods

## Employee Documentation

### Offer Letters / Employment Agreements
- [ ] All employee offer letters
- [ ] Employment agreements (if used)
- [ ] Compensation records

**For each employee verify:**
- Written offer letter/agreement exists
- At-will status stated (in at-will states)
- Compensation matches records
- Start date documented

### Employee Confidentiality & IP
- [ ] PIIA / CIIAA for all employees
- [ ] Confidentiality agreements
- [ ] Non-compete agreements (if enforceable)
- [ ] Non-solicitation agreements

**Verify signed before first day of work**

### Employee Handbook
- [ ] Employee handbook
- [ ] Acknowledgment signatures
- [ ] Anti-harassment policy
- [ ] Equal opportunity policy

## Contractor/Consultant Arrangements

### Contractor Agreements
- [ ] Independent contractor agreements
- [ ] Consulting agreements
- [ ] Statements of work

**For each contractor verify:**
- Written agreement exists
- IP assignment included
- Proper classification (not misclassified employee)
- 1099 issued (if applicable)

### Classification Analysis
- [ ] Contractor vs. employee classification review
- [ ] IRS factors analysis
- [ ] State law compliance (CA AB5, etc.)

**Misclassification risk factors:**
- Works full-time/exclusively for company
- Uses company equipment
- Company controls how work is done
- No own business entity
- Long-term engagement

## Key Employee Matters

### Key Persons
- [ ] Identify key employees
- [ ] Employment agreement status
- [ ] Non-compete enforceability
- [ ] Flight risk assessment

### Departures
- [ ] Termination letters (departed employees)
- [ ] Separation agreements
- [ ] Release of claims
- [ ] Return of property confirmation

**For any contentious departures:**
- Circumstances documented
- Legal counsel involved
- Claims released (if applicable)

## Benefits & Compensation

### Benefit Plans
- [ ] Health insurance plans
- [ ] 401(k) or retirement plans
- [ ] Other benefits (dental, vision, etc.)
- [ ] Plan documents and SPDs

### Equity Compensation
- [ ] Option grant agreements
- [ ] RSU agreements (if any)
- [ ] Exercise documentation

### Wage and Hour
- [ ] Exempt vs. non-exempt classification
- [ ] Overtime policies
- [ ] Timekeeping records (non-exempt)
- [ ] State wage law compliance

## HR Compliance

### Required Notices/Postings
- [ ] Federal labor law postings
- [ ] State labor law postings
- [ ] Workers compensation notice

### I-9 Compliance
- [ ] I-9 forms for all employees
- [ ] E-Verify participation (if required)

### Background Checks
- [ ] Background check consent forms
- [ ] FCRA compliance
