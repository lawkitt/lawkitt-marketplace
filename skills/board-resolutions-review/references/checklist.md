## Checklist Items

### 1. Document Completeness
- [ ] Full resolution document available
- [ ] All exhibits/attachments identified
- [ ] Referenced documents attached or identified

### 2. Entity Identification
- [ ] Company name and state of incorporation
- [ ] Entity type (corporation, LLC)
- [ ] If DE corp: DGCL applies

### 3. Approving Body
- [ ] Correct body: Board vs Stockholders vs Committee
- [ ] Action within body's authority
- [ ] If committee: acting within delegated scope

### 4. Director/Member List
- [ ] Names of all directors listed
- [ ] "Constituting all directors" or quorum stated
- [ ] Number matches current board composition

### 5. Quorum
- [ ] Quorum present or stated
- [ ] Majority of directors (or as bylaws specify)
- [ ] If written consent: unanimous or as permitted

### 6. Voting Threshold
- [ ] Appropriate threshold for action type
- [ ] Majority of quorum (typical) or supermajority if required
- [ ] Class voting if applicable (preferred stock)

### 7. Conflicts of Interest
- [ ] Interested directors disclosed
- [ ] Interested directors recused or waived
- [ ] Disinterested quorum obtained (if required)
- [ ] Transaction fair to company (if interested)

### 8. Effective Date
- [ ] Effective date stated
- [ ] Clear: specific date or "upon execution"
- [ ] Time-sensitive actions: date critical

### 9. Notice and Waiver
- [ ] Meeting notice given OR waiver obtained
- [ ] Waiver statement included (if no meeting)
- [ ] If written consent: timing complies with DGCL ยง228

### 10. Transaction Identification
- [ ] Documents being approved specifically identified
- [ ] Parties, dates, amounts stated
- [ ] Attached as exhibits OR "substantially in form presented"

### 11. Securities Authorization (if applicable)
- [ ] Number and class of shares authorized
- [ ] Consideration stated (DGCL ยง152)
- [ ] Per-share price or valuation basis
- [ ] 409A valuation referenced (for options)

### 12. Officer Delegation
- [ ] Authority granted to specific officers
- [ ] Scope limited to transaction
- [ ] Materiality qualifiers (if broad delegation)

### 13. Minute Book Instruction
- [ ] Resolution to be filed in minute book
- [ ] Secretary certification contemplated

### 14. Signature Blocks
- [ ] All required signatures identified
- [ ] Title and capacity stated
- [ ] Date lines present
